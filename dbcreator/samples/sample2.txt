The university keeps track of each student's name, unique student number, Social Security number, current address and phone number, permanent address,
phone number, birth date, sex, class, major department, minor department and degree program.
Some user applications need to refer to the city, state, and ZIP Code of the permanent address and to the student's last name.
Each student has a unique Social Security number.
Each department is described by a name, department code, office number, office phone number, and college.
Each department has a unique name and unique ZIP code.
Each course has a course name, description, course number, number of semester hours, level, and department.
The value of the course number is unique for each course.
Each section has an instructor, semester, year, course, and section number.
The section number distinguishes sections of the same course that are taught during the same semester/year.
A grade report has a student, section, letter grade, and numeric grade.
